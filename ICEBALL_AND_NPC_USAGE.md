# 冰球术技能和无敌NPC使用说明

## 冰球术技能

我们已经成功添加了一个新的冰系技能——冰球术(ice.iceball)。

### 技能属性
- 名称: 冰球术
- 技能键名: ice.iceball
- 基础伤害: 40
- 每级增加伤害: 8
- MP消耗: 18
- 施法时间: 1秒
- 冷却时间: 4秒
- 作用范围: 10米
- 描述: 凝聚寒冰向目标投出一枚冰球。

### 使用方法
1. 首先学习技能:
   ```
   learn ice.iceball
   ```
   或
   ```
   learn 冰球术
   ```

2. 查看已学技能:
   ```
   skills
   ```

3. 对目标使用技能:
   ```
   cast ice.iceball <目标名称>
   ```
   或
   ```
   cast iceball <目标名称>
   ```

## 无敌NPC功能

我们创建了一个特殊的NPC类型，可以设置为无敌模式。

### 创建普通NPC
```
create_npc <NPC名称>
```
例如:
```
create_npc 僵尸
```

### 创建无敌NPC
```
create_npc <NPC名称> = invincible
```
例如:
```
create_npc 僵尸 = invincible
```

### 无敌NPC的特点
- 不会因受到伤害而减少HP
- 当被攻击时会显示"你的攻击对这个目标无效"
- 攻击者会收到提示"你的攻击对<NPC名称>无效"

### 测试建议
1. 创建一个普通NPC和一个无敌NPC
2. 使用火球术和冰球术分别攻击它们
3. 观察普通NPC会受到伤害，而无敌NPC不会受到伤害

这样你就可以测试技能系统在面对不同类型敌人时的表现了。
