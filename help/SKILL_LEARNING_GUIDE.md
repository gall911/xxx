# 技能学习指南

## 问题分析

您在尝试学习冰球术技能时遇到了"找不到技能：iceball"的错误。经过检查，我们发现：

1. 冰球术技能文件 (`world/magic/ice/iceball.json`) 已正确创建
2. 技能索引文件 (`world/magic/magic_index.json`) 已包含冰球术的条目
3. 技能学习命令 (`commands/skill_learn.py`) 已实现

## 解决方案

### 方法一：使用完整技能键名学习

在游戏内使用完整技能键名学习冰球术：

```
learn ice.iceball
```

### 方法二：使用技能中文名称学习

您也可以使用技能的中文名称来学习：

```
learn 冰球术
```

## 技能施放

学习技能后，您可以使用以下命令施放冰球术：

```
cast ice.iceball <目标>
```

或者：

```
iceball <目标>
```

## 验证技能是否学会

您可以使用以下命令查看已学会的技能：

```
skills
```

## 常见问题

1. **找不到技能**：确保使用正确的技能键名或中文名称
2. **权限问题**：确保您的角色有足够权限学习技能
3. **服务器未重启**：如果最近添加了新技能，可能需要重启服务器

## 技能列表

当前可用的冰系技能：
- `ice.iceball` - 冰球术
- `ice.iceblast` - 冰爆术

火系技能：
- `fire.fireball` - 火球术
- `fire.firewall` - 火墙术

武学技能：
- `sword.slash` - 斩击
